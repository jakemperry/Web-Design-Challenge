Summary: Weather Trends Across Latitudes

How does temperature change as you get closer to the equator? Does humidity change with latitude? Where's the weather the most windy? Are grey skies turned upside down if you're Down Under?

This gallery shows results from a python API exercise, where weather data were collected uising the Open Weather Map API based on city locations in the CitiPy library. After collecing weather data from a set of random cities, the data were plotted using matplotlib.

Please keep in mind: the weather data were collected on 7 February 2021, so many of the weather patterns in the data may reflect winter conditions in the Northern Hemisphere and summer conditions in the Southern Hemisphere. This project may be updated later to evaluate weather trends over time to see how different seasons may affect the conditions at different latitudes.
